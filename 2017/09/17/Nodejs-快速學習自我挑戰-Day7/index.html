  1. Postman 官網

Resource Creation Endpoint - POST /todos

  1. 新增 server/db/mongoose.js
  1. 新增 server/models/todo.js
  1. 新增 server/models/user.js
  1. 剛剛把 server/server.js 的資料都搬到了其它資料夾，所以檔案簡化為
  1. 安裝 body-parse 可以把 JSON 轉換為 object
    npm i express@4.14.0 body-parser@1.15.2 --save
  2. 在 server/server.js 引入 Library
  1. 使用 express
  1. 在 Postman 新增 Post http://localhost/todos，選擇 Body，選擇 raw，選擇 JSON，並送出 Post

測試 POST /todos

    module.exports = {app};

    • 也加入新的 collection
