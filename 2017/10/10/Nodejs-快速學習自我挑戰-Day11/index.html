Nodejs 快速學習自我挑戰 Day11

2017-10-10
  • Nodejs

使用 Socket.io 的即時 Web Apps

Broadcasting Events

  1. 刪除 server/server.js 和 public/js/index.js 的 socket.emit
  2. 在 server/server.js 新增 io.emit
  1. git push heroku master
  2. 在 server/server.js 新增兩種 emit，如果 connect 就傳送歡迎，如果有新用戶進來，就 broadcast 新用戶加入

訊息產生器和測試

  1. 新增 server/utils/message.js
  1. npm install expect mocha --save-dev
  2. 修改 package.json
  1. 新增 server/utils/message.test.js
  2. npm test
  3. 修改 server/utils/message.test.js
  1. 在 server/server.js 引入 const {generateMessage} = require('./utils/message');
  2. 將 server/server.js 的 object 換成 function

Event Acknowledgements

  1. 在 public/js/index.js 新增 emit
  1. 在 server/server.js 新增 callback function

訊息表單和 jQuery

  1. 下載最新版的 jQuery
  2. 放到 public/js/libs/jquery.js
  3. 在 public/index.html 引入 <script src="/js/libs/jquery-3.2.1.min.js"></script>
  4. 在 public/index.html 新增 form
  1. 在 public/js/index.js 新增 jQuery 來取得 form 傳出來的訊息
  1. 在 public/index.html 新增 <ol id="messages"></ol> 讓傳出得的訊息 console 在頁面上
  2. 在 public/js/index.js 將取得的文字用 jQuery 創建元素並顯示在前端
  1. git push heroku master

Geolocation

  1. Geolocation 官方文件
  2. 在 public/index.html 新增 button
    <button id="send-location">Send Location</button>
  3. 在 public/js/index.js 新增 click 事件，當點擊時傳送出 location 的資訊
  1. 在 server/server.js 新增 newMessage 把訊息傳到前端頁面
  1. 在 server/server.js 新增新的 message，讓 locaiton 可以超連結
  1. 在 server/utils/message.js 新增 generateLocationMessage 並 export
  1. 在 public/js/index.js 將取得的 location 傳送到前端頁面
  1. 在 server/utils/message.test.js 測試 generateLocationMessage 可以運作
  1. git push heroku master

裝飾聊天頁面

  1. 樣式
  2. 新增 public/css/styles.css
  1. 在 public/index.html 引入 <link rel="stylesheet" href="/css/styles.css">
  2. 在 public/index.html 加入樣式
  1. 在 server/server.js 把 callback 改為空值
  1. 修改 public/js/index.js 用 jQuery 取得值，然後送到 server
  1. 修改 public/js/index.js 當送出的時候把按鈕變成無法點擊，完成之後就移除無法點擊的效果

Timestamps 和使用 Moment 格式化

  1. npm i moment --save
  2. Momentjs 官方網站
  3. 新增 playground/time.js 測試 moment.js

Printing Message Timestamps

  1. 修改 server/utils/message.js，引入 moment，然後將 createdAt 改為 moment().valueOf();
  1. npm test，確定測試可以通過
  2. 從 node_modules/moment/moment.js 複製檔案到 public/js/libs/moment.js
  3. 在 public/index.html 引入 <script src="/js/libs/moment.js"></script>
  4. 讓訊息送出時，同時顯示時間
